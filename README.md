# Zeke Bank 

A command-line banking system built in Python.

## Features
- Create account with unique 12-digit account number
- PIN encryption using SHA-256
- Temporary 30-minute account block after 3 wrong PIN attempts
- Deposit, withdraw, and transfer funds
- Real-time transaction history (opens as CSV)
- PIN reset
- Cross-platform (Windows, Mac, Linux)

## How It Works
Accounts are stored locally as JSON files.
Transactions are logged to a CSV file per user.
Transfers update both sender and receiver transaction history.

## Built With
- Python standard library only
- hashlib (PIN encryption)
- json (data storage)
- csv (transaction logging)
